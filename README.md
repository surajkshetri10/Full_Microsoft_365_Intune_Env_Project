Suraj365lab Solutions – Modern Workplace Project
A complete Microsoft 365 + Intune enterprise environment built from scratch.

Overview
This project simulates the work of an IT Support Technician in a fictional company called Suraj365lab Solutions.  
The goal was to design, build, and manage a full modern workplace environment using:
• Microsoft 365
• Entra ID (Azure AD)
• Intune (Endpoint Manager)
• Exchange Online
• SharePoint Online
• Microsoft Teams
The environment mirrors how real organisations manage identity, devices, security, collaboration, and user lifecycle workflows.

Project Objectives
This project includes:
• Building a complete Microsoft 365 tenant
• Creating users, departments, and security groups
• Assigning licenses and roles
• Configuring MFA & Conditional Access
• Deploying Intune and enrolling Windows devices
• Creating compliance & configuration policies
• Deploying apps automatically to devices
• Building SharePoint sites and Teams channels
• Designing onboarding & offboarding workflows
• Troubleshooting real-world issues
• Documenting everything professionally
